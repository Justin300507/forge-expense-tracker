# Forge Expense Tracker

> Generated by **ForgeAI** — AI-powered full-stack application generator.

## Features

- Allows users to record individual expenses with amount, description, and category.
- Enables users to create, view, and assign categories to their expenses.
- Permits users to define monthly budget limits for specific expense categories.
- Visualizes spending by category against set budgets for a given period (e.g., current month).
- Displays a chronological list of all recorded expenses, with filtering by category.

## Quick Start

### Prerequisites

- Python 3.11+
- Node.js 18+ (for frontend)

### Backend

```bash
cd app
pip install -r requirements.txt
uvicorn main:app --reload
```

API available at: `http://localhost:8000`
Interactive docs: `http://localhost:8000/docs`

### Frontend

```bash
npm install
npm run dev
```

Frontend at: `http://localhost:5173`

### Docker

```bash
docker build -t forge_expense_tracker .
docker run -p 8000:8000 forge_expense_tracker
```
